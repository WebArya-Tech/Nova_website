# Nova Tuitions Website

## Project Info

This project is a Vite + React + TypeScript website for Nova Tuitions.

## Run Locally

Requirements:
- Node.js 18+
- npm or bun

Commands:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
